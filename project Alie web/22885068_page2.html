CSS Examples

Selectors are important in using CSS because that identify which element of HTML you want to style. Then, The CSS can be done by adding the style attributes such as (background-color; font-weight; color...)inside the style area.

  1. Simple Selectors
    Selectors code Descript Examples
    element selector p { text-align: center; color: red; } All the content inside the p <tag>will be styled text center and color red

    looking example
    id Selector # id { background_color: yellow; } All id element with the name of id will style by yellow background looking example
    class Selector .center { color: green; } All element with class named center will style fornt color by green. looking example
  2. Combinator selector
    Selectors Code Descript Example
    descendant selector div p { color: red; } All p tag of the descendant of div tag will be styled

    looking example
    child selector div > p { background-color: yellow; } using >, only the child tag of the father tag can be styled. looking example
    Adjacent Sibling Selector div + p { background-color: yellow; } The p element that is directly after <div> element will be styled. looking example
    General Sibling Selector div ~ p { background-color: yellow; } selects all elements that are siblings of a specified element looking example
  3. Pseudo-class selectors
    Syntax Example Description All CSS Pseudo Classes
    selector:pseudo-class { property: value; } This is a link The Pseudo-classes of link,visited,hover and active are used in this example to change color when the link haven't visited, after visted, hover over and be active
  4. Pseudo-elements selectors
    Syntax Example Descript All CSS pseudo-element
    selector::pseudo-element { property: value; }

  5. Attribute selectors
    Code Example Descript All CSS Attribute Selectors
    [title~="flower"] { border: 5px solid yellow; } the content of the title with letters of flower are styled.

    window.onload = function() { var myNavBar = document.getElementById('myNavBar') window.addEventListener('scroll', function() { var targetTop = window.pageYOffset if (targetTop > 240){ myNavBar.classList.add('fixed-top') }else{ myNavBar.classList.remove('fixed-top') } }) }

    This function is a global function. we define the content of navigation bar as variable of myNavBar. We define a variable called targetTop that is the scoll distance of current web page. If the distance greater than 240, We let the myNavBar move to the top area of current web location, otherwise the myNavBar keep in current location.
